# RuMedia Release Details Helper

Пользовательский скрипт Tampermonkey, который добавляет быстрый просмотр деталей релиза и модераторских комментариев прямо в очереди релизов RuMedia.

## Возможности
- Кнопка **«Посмотреть детали»** раскрывает автора инструментала и тип вокала из формы редактирования трека.
- Кнопка **«Наличие комментов»** загружает модераторские комментарии к треку или альбому и выводит их во всплывающем окне.
- Комментарии показываются с дружелюбными именами модераторов (moderator3 → «Руслан», moderator7 → «Матвей»).
- Время отображается в относительном и абсолютном формате: «N часов назад (ДД.ММ.ГГГГ ЧЧ:ММ)» даже если на странице нет атрибута `title`.
- Все данные кэшируются в рамках сессии, чтобы не дергать одни и те же страницы повторно.

## Установка
1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) в браузер.
2. Откройте файл [`userscripts/rumedia-release-info.user.js`](userscripts/rumedia-release-info.user.js) прямо из репозитория.
3. Нажмите **Install** в интерфейсе Tampermonkey.

## Использование
1. Перейдите на страницу очереди: `https://rumedia.io/media/admin-cp/manage-songs` или `https://rumedia.io/media/admin-cp/manage-albums`.
2. Рядом с кнопкой «Добавить в очередь» появятся две новые кнопки:
   - **Посмотреть детали** — подгружает автора инструментала и тип вокала.
   - **Наличие комментов** — проверяет, есть ли комментарии модераторов, и выводит их с датой/временем.
3. При повторных нажатиях данные берутся из кэша, что ускоряет работу и снижает нагрузку на сервер.
